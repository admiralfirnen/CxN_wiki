# Screenshots Folder

This folder is for storing game screenshots used throughout the wiki.

## Naming Convention

Use `kebab-case` for all filenames:
- `ragnarok-event-map.png`
- `hero-equipment-guide.jpg`
- `clan-war-results-2026-01.png`

## Organization Tips

Consider prefixing by category:
- `event-*` - Event screenshots
- `guide-*` - Tutorial/guide images
- `ui-*` - Game UI elements
- `battle-*` - Battle screenshots
