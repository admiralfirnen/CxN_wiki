Put your downloadable files here (images, PDFs, etc.).

Add entries in data/downloads.json with this format:

  {
    "title": "Display name",
    "file": "assets/downloads/yourfile.png",
    "thumbnail": "assets/downloads/yourfile.png",
    "description": "Optional short description"
  }

- "file" is required (path relative to site root).
- "thumbnail" is optional; for images, omit to use the file as preview.
- "description" is optional.
